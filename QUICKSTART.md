## Quick Start (English)

### 1. Install

```bash
npm install
```

### 2. Configure

Copy `.env.example` to `.env` and add your API key:

```bash
cp .env.example .env
```

### 3. Run

```bash
npm run dev
```

### 4. Open Browser

- Root URL (auto-redirects to Russian): http://localhost:3000
- Russian version: http://localhost:3000/ru
- English version: http://localhost:3000/en
